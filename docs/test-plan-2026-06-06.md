## 二、端到端测试场景

### E2E-01: 首页加载 → 选择路线 → 查看课时 → 完成标记

```powershell
# Step 1: 验证首页加载
$r = Invoke-WebRequest "http://localhost:3000/" -UseBasicParsing
assert $r.StatusCode -eq 200
assert $r.Content.Contains("AI 开发者实训平台")
assert $r.Content.Contains("Agent 开发工程师")

# Step 2: 获取路线数据
$tracks = Invoke-RestMethod "http://localhost:3000/api/tracks"
assert $tracks.Count -ge 1
assert $tracks[0].slug -eq "agent-engineer"

# Step 3: 进入路线详情
$track = Invoke-RestMethod "http://localhost:3000/api/tracks/agent-engineer"
assert $track.modules.Count -ge 6  # 原有6个 + 新增4个 = 10个
assert $track.modules[0].lessons.Count -ge 3

# Step 4: 进入课时播放器
$modId = $track.modules[0].id
$lessonSlug = $track.modules[0].lessons[0].slug
$r = Invoke-WebRequest "http://localhost:3000/learn/agent-engineer/$modId/$lessonSlug" -UseBasicParsing
assert $r.StatusCode -eq 200

# Step 5: 标记课时为完成
$body = @{ lessonId = $track.modules[0].lessons[0].id; status = "completed" } | ConvertTo-Json
$r = Invoke-WebRequest "http://localhost:3000/api/progress" -Method PUT -Body $body -ContentType "application/json" -UseBasicParsing
assert $r.StatusCode -eq 200

# Step 6: 验证进度已更新
$progress = Invoke-RestMethod "http://localhost:3000/api/progress"
assert ($progress | Where-Object { $_.status -eq "completed" }).Count -ge 1
```

**预期结果**: 
- 首页正常渲染，展示路线卡片
- 路线详情展示所有模块和课时
- 课时页加载 MDX 内容
- 完成标记成功，进度可查询

## 六、性能基准

| 指标 | 目标 | 实测 |
|------|------|------|
| 首页 FCP | < 1.5s | |
| 首页 LCP | < 2.5s | |
| 首页 JS Bundle | < 200KB | 146KB |
| API 响应时间 (tracks) | < 100ms | |
| 图谱节点加载 (50个) | < 500ms | |
| 数据库查询 (tracks) | < 10ms | |
| 课时页首屏 | < 2s | |

```powershell
# 性能测试脚本
$times = @()
for ($i = 0; $i -lt 10; $i++) {
    $sw = [System.Diagnostics.Stopwatch]::StartNew()
    $null = Invoke-WebRequest "http://localhost:3000/api/tracks" -UseBasicParsing
    $sw.Stop()
    $times += $sw.ElapsedMilliseconds
}
$avg = ($times | Measure-Object -Average).Average
Write-Host "API avg response time: $($avg)ms"
```
