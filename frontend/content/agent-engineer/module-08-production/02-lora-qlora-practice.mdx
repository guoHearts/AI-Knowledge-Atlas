# LoRA 与 QLoRA 实战——单卡微调 70B 模型

> 💡 **单张消费级显卡微调 70B 模型** — LoRA + 4-bit 量化让微调不再需要数据中心。

## LoRA 关键参数

| 参数 | 含义 | 建议值 |
|------|------|--------|
| **r** (rank) | 低秩矩阵的秩 | 8-64，越大效果越好但越慢 |
| **alpha** | 缩放因子 | 通常 = 2×r |
| **target_modules** | 哪些层加 LoRA | q_proj, v_proj（注意力层） |
| **dropout** | 正则化 | 0.05-0.1 |

## QLoRA 实战代码

使用 Unsloth（最快最省显存）：

```python
# 安装：pip install unsloth
from unsloth import FastLanguageModel
from datasets import load_dataset
from transformers import TrainingArguments
from trl import SFTTrainer

# ── 第 1 步：加载 4-bit 量化模型 ──
model, tokenizer = FastLanguageModel.from_pretrained(
    model_name="unsloth/Meta-Llama-3.1-8B-bnb-4bit",  # 4-bit 预量化
    max_seq_length=2048,
    load_in_4bit=True,
)

# ── 第 2 步：添加 LoRA 适配器 ──
model = FastLanguageModel.get_peft_model(
    model,
    r=16,                     # LoRA rank
    target_modules=[          # 只训练这些层
        "q_proj", "k_proj", "v_proj", "o_proj",
        "gate_proj", "up_proj", "down_proj",
    ],
    lora_alpha=16,
    lora_dropout=0,
    use_gradient_checkpointing="unsloth",  # 省显存
)

# ── 第 3 步：准备训练数据 ──
# 数据格式（Alpaca 格式）
training_data = [
    {
        "instruction": "将以下中文翻译为英文",
        "input": "人工智能正在改变世界",
        "output": "Artificial intelligence is changing the world.",
    },
    # ... 至少 100-1000 条
]

def format_prompt(example):
    return f"""### Instruction:
{example['instruction']}

### Input:
{example['input']}

### Response:
{example['output']}"""

# ── 第 4 步：训练 ──
trainer = SFTTrainer(
    model=model,
    tokenizer=tokenizer,
    train_dataset=dataset,
    formatting_func=format_prompt,
    args=TrainingArguments(
        per_device_train_batch_size=2,
        gradient_accumulation_steps=4,
        warmup_steps=5,
        max_steps=100,           # 小数据集不需要太多步
        learning_rate=2e-4,
        fp16=not model.get_memory_footprint() > 20_000_000_000,
        logging_steps=10,
        output_dir="outputs",
    ),
)

trainer.train()

# ── 第 5 步：保存模型 ──
model.save_pretrained_merged("my-lora-model", tokenizer, save_method="merged_16bit")
```
