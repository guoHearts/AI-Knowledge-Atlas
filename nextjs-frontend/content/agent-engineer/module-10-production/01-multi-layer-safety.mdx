# 多层安全防护体系

> 💡 **安全不是一道门，是层层防护** — 任何单层都可能被突破，多层才能兜底。

## 四层防护架构

```
用户输入
    │
┌───▼──────────────┐
│ 第 1 层：输入护栏  │  ← 关键词过滤、模式检测、内容安全模型
└───┬──────────────┘
    │ 过滤后
┌───▼──────────────┐
│ 第 2 层：上下文净化│  ← 剥离可疑指令、系统提示隔离
└───┬──────────────┘
    │
┌───▼──────────────┐
│ 第 3 层：Agent 执行│  ← 最小权限原则、Tool 权限分级
└───┬──────────────┘
    │
┌───▼──────────────┐
│ 第 4 层：输出护栏  │  ← 敏感信息脱敏、有害内容过滤
└───┬──────────────┘
    │
  最终输出
```

## 第 1 层：输入护栏

```python
from transformers import pipeline

# 使用 Llama Guard 做内容安全分类
safety_classifier = pipeline(
    "text-classification",
    model="meta-llama/Llama-Guard-3-8B",
)

UNSAFE_CATEGORIES = {
    "S1": "暴力犯罪",
    "S2": "色情内容", 
    "S3": "仇恨言论",
    "S7": "Prompt 注入",
    "S10": "越狱尝试",
}

def input_guard(user_input: str) -> dict:
    """输入内容安全检测"""
    result = safety_classifier(user_input)[0]
    
    if result['label'] != 'safe':
        category = UNSAFE_CATEGORIES.get(
            result['label'].split('\n')[0], 
            result['label']
        )
        return {
            "safe": False,
            "category": category,
            "action": "block",
            "message": f"输入包含不安全内容（{category}），已被阻止。",
        }
    
    return {"safe": True}
```

## 第 4 层：输出护栏

```python
def output_guard(response: str) -> str:
    """输出安全检测和脱敏"""
    
    # 1. 敏感信息脱敏
    SENSITIVE_PATTERNS = [
        (r'sk-[a-zA-Z0-9]{32,}', '[API_KEY_REDACTED]'),
        (r'[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}', '[EMAIL_REDACTED]'),
        (r'\b\d{16,19}\b', '[CARD_NUMBER_REDACTED]'),
    ]
    
    for pattern, replacement in SENSITIVE_PATTERNS:
        response = re.sub(pattern, replacement, response)
    
    # 2. Llama Guard 输出安全检测
    safety_result = safety_classifier(response)[0]
    if safety_result['label'] != 'safe':
        return "抱歉，生成的回复包含不安全内容，已被拦截。"
    
    return response
```
